untracked são arquivos q o git não tem ciência dele
tracked são arquivos q o git tem ciência dele

modified é um arquivo que foi modificado

o que são repositórios? 

o git é um sistema distribuido

os arquivos vão ficar transitando entre working directory para a staging area

quando você faz os arquivos entrar em cena você move os arquivos da área de staging para commit e então eles voltam para o unmodified
e voltam para este ciclo

então você popula o seu local repository
tudo que estiver lá tem que estar "commitado"
